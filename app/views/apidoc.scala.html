JavaScript API

Recording

Just enter in your source page the following Scriptlet: 
<script type="text/javascript" src="http://video-rec.lean-web-development.com/yoursite/record.js"/>
<script type="text/javascript">
	LWDRecord.button({});
</script>

Replace yoursite with your registered application name. This will display the record button. On click, this button will load our recording widget on demand and ask the user for starting her recording.

The argument to the button is a configuration object, consider: 

	LWDRecorder.button({
		button : {
			title : "Start recording your comment"
		}
	});
Here is the full list of configuration options:
Object Parameter Default Description
button title Record... Title of the recording button
meta key A key you want this recorded video be stored under. Later, you can query for this key.

Replay

Replay in fact is a query for recorded video. The simpliest way is: 
<script src="http://video-rec.lean-web-development.com/yoursite/replay.js" type="text/javascript"/>
<script type="text/javascript">
	LWDRecord.replay({});
</script>
Replace yoursite with your registered application name. This will display the record button. On click, this button will load our recording widget on demand and ask the user for starting her recording.
